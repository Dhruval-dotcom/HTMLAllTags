Button tag

The <button> tag defines a clickable button.

Inside the form the attributes related to button are
Attribute Value Description
autofocus autofocus Specifies that a button should automatically get focus when the page loads
disabled disabled button should be disabled
form form_id which form the button belongs to
formaction URL where to send the form-data when a form is submitted. Only for type="submit"
formmethod get
post		 how to send the form-data (which HTTP method to use). Only for type="submit"
formtarget _blank
_self
_parent
_top
framename 		Specifies where to display the response after submitting the form. Only for type="submit"
name name Specifies a name for the button
type button
reset
submit		 Specifies the type of button
value text Specifies an initial value for the button
